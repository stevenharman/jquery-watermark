jQuery Watermark Plugin

v 1.1.1

This jQuery plugin will easily create watermark hints in input elements.

Features:

  • Watermark style is controlled by CSS classes
  • Each input element can have its own independent watermark class/style
  • Each input element can have its own watermark text
  • Plugin automatically removes all watermarks prior to form submission
  • Both watermark text and CSS class name can be changed dynamically
  • Visual Studio Intellisense-compatible documentation included in source
  • Comes with jQuery selector extension ":data(...)" (documented in source)
  • Minified version very compact — about 2,000 bytes
